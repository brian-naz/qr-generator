# QR Code Generator

To get started:
<a href="https://qr-generator-brian-naz.vercel.app/">Demo</link>

### Setup:

Clone the repo:
```bash
git clone https://github.com/brian-naz/qr-generator.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
